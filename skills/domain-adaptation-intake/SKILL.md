# 新数据源准入

## 何时调用

- 同事拿到一批非访谈类文本，想沿用 Stream 2 方法  
- 需要决定「是否拆句」「是否分层」「双路是否全开」  

## 执行步骤

1. 打开并共同填写 `methodology-kit/domain_adaptation_intake.md`。  
2. 空项不得超过 3 个；关键空项（分析单位、研究问题、偏差）必须补齐才能开编码。  
3. 将填好的文件保存到具体项目目录，例如 `docs/domain_adaptation_intake.md`，并在 README 中链接。  

## 与仓库的衔接

- **Stream 2** 参考：`stream2-analysis/docs/methodology_framework.md`  
- **Stream 1A** 参考：`stream1a/README.md`（含数据陷阱）  

## 输出

一份填完的 `domain_adaptation_intake.md` + 口头结论三句：**能不能做 / 先做试点还是全量 / 最大风险是什么**。
